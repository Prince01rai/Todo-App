# Todo-App
A simple and modern To-Do Application built using Vue.js (Frontend) and JSON-Server.  Features include task creation, editing, deletion, completion status, filters (All, Open, Completed),  and dark/light mode UI. This project is submitted as part of the FWeb Developer assignment for AptCoder.
